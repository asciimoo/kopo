    OTR

      • Pidgin + OTR plugin

    • MacOS:
      • Adium (OTR plugin included. Remember to disable logging)
      • mcabber - small and better code quality, no GUI though

    Tails

    It is a complete operating-system designed to be used from a CD or a USB stick independently of the computer's original operating system. It is Free Software and based on Debian GNU/Linux. Tails contains the tools listed above preconfigured in a secure fashion. Get Tails here
